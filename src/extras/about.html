About Todoist for Linux (v)

Todoist is a To-do app used by many people. This app is an open-source implementation of the Todoist Web App to create a Linux native package.

Disclaimer: This app is community made and not an official release.

Original Author: Ruslan Bekenev (https://github.com/krydos)
