About Todoist for Linux

Todoist is a productivity app used by millions around the world. This is an open-source application that wraps the official Todoist Web App using the popular and open-source Electron library.

This is NOT an official product. This project does not attempt to reverse engineer the Todoist Web Application (or API) or attempt to reimplement any part of the Todoist web app. Any interaction between the user and Todoist Web app is handled by the Todoist Web app itself; this is just a native wrapper for Todoist Web, like a browser, built for Linux only.
